# Frontend Mentor Project 05 - Clipboard Landing Page

## Overview
This is a responsive landing page for the **Clipboard** app, built using **HTML** and **CSS**. The page showcases app features, provides download links, and includes company logos and social media links.



## Features
- **Responsive design**: Optimized for mobile and desktop screens.
- **CTA sections**: Prominent call-to-action buttons for app downloads.
- **Feature highlights**: Sections to display app capabilities and workflow benefits.
- **Social media links**: Footer includes clickable FontAwesome icons.
- **Company logos**: Trusted brands displayed to increase credibility.
- **Custom fonts**: Bai Jamjuree Light, Medium, and Bold.

## Technologies Used
- HTML5
- CSS3
- [FontAwesome](https://fontawesome.com/) icons
- Custom fonts (Bai Jamjuree)

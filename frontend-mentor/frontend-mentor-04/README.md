# Frontend Mentor Project 04 - Coming Soon Landing Page

## Overview
This is a responsive "Coming Soon" landing page project built using **HTML**, **CSS**, and minimal JavaScript for form interactions. The page is designed to notify users when a website or service is launching and to collect email subscriptions.



## Features
- **Responsive layout**: Optimized for mobile and desktop screens.
- **Subscription form**: Input field with validation and notification for users.
- **Modern styling**: Rounded inputs and buttons, subtle shadows, and color scheme based on blue and gray tones.
- **FontAwesome integration**: Social media icons in the footer.
- **Custom fonts**: Uses Libre Franklin and Libre Franklin Italic for typography.

## Technologies Used
- HTML5
- CSS3
- JavaScript (for form validation)
- [FontAwesome](https://fontawesome.com/) icons
- Google Fonts: Libre Franklin

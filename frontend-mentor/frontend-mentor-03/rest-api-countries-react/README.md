# Rest API Countries

A responsive React application that fetches country data from the [REST Countries API](https://restcountries.com/) and displays it in a searchable and filterable interface. Includes light/dark mode toggle and detailed country pages.


## Technologies Used

- React
- React Router DOM
- CSS
- FontAwesome Icons

## Features

- Light/Dark mode toggle
- Search countries by name
- Filter countries by region
- Country detail page with:
  - Native Name
  - Population
  - Region & Subregion
  - Capital
  - Top Level Domain
  - Currencies
  - Languages
- Responsive layout for mobile and desktop

## Overview
This is a responsive landing page built as a solution for the Frontend Mentor challenge. The project focuses on clean design, accessibility, and responsiveness across devices.

### Key Features
- Responsive design for mobile, tablet, and desktop
- Smooth navigation menu with hover effects
- Feature section with icons and descriptive text
- Testimonials section showcasing team feedback
- Contact form with input validation (JS)
- Footer with location, links, and social media integration

## Technologies
This project uses:
- HTML5
- CSS3 (Flexbox and CSS variables)
- JavaScript (for simple form interactions)
- FontAwesome for social and UI icons
- Custom fonts: Raleway and Open Sans
